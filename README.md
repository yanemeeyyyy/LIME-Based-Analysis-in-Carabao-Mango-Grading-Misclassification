# LIME-Based-Analysis-in-Carabao-Mango-Grading-Misclassification

•	This research makes use of AI for both classification (YOLOv8) and interpretation (LIME) in the decision-making of the deep learning algorithm used in classification and the user interface was developed using Tkinter-based libraries through Python.

•	Built a feedback mechanism where users validate predictions as correct or incorrect; incorrect predictions proceed to LIME analysis (cropping image -> removing background image -> selecting correct class -> selecting image instance for LIME analysis) to provide visual explanations of the model’s behavior in its decision making.

•	Implemented logging functionality to a CSV file (correct and incorrect) to record YOLOv8 prediction results.
